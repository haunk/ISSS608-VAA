## 🚀 Getting Started

To run this site locally:

1.  **Clone the repository**

``` bash
git clone https://github.com/haunk/ISSS608-VAA.git
cd ISSS608-VAA
```

2.  **Open in RStudio** and install Quarto if not already installed: [quarto.org](https://quarto.org/docs/get-started/)

3.  **Preview the site**

``` bash
   quarto preview
```

4.  **Render the site**

``` bash
   quarto render
```
